# Use Meson

You need ==Meson== to compile the project,
after installing it and it's dependencies
use the code below

==DO NOT FORGET== hydrate yourself :joy

```bash
meson setup build
cd build
meson compile
```


## == // Dependencies // ==

```txt
sdl3
sdl3_ttf
sdl3_image
glfw3
vulkan-headers
opencl-headers
```
