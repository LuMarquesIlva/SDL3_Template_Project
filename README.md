# Use Meson

You need __Meson__ to compile the project,
after installing it and it's dependencies
use the code below

__DO NOT FORGET__, hydrate yourself 🍉

```bash
meson setup build
cd build
meson compile
```


## -- // Dependencies // --

```txt
sdl3
sdl3_ttf
sdl3_image
glfw3
vulkan-headers
opencl-headers
```
